JavaScript OOP: Carousel From Scratch

Below are four Carousel objects created from the same Carousel class. Basic features are as follows:

  • Viewing size is 400x300 pixels
  • Navigation arrows appear at side edges on hover
  • Download is on lower-right corner
  • Pause/unpause on middle hover
  • Fetches JSON via AJAX
  • Wraps around
  • Auto-sliding direction follows the last navigation arrow pressed (i.e., if you pressed the left arrow, the carousel will auto-slide towards the left.)
  • Default transition time for auto-slide is 1800ms

Default Carousel

This carousel uses all basic features mentioned. All images are 400x300 in size.

Support for User-configured Options

Adjustable options include: viewing size (600x200), Background-color (pink), & Transition time (4000ms). All images shown are 400x300 in size.

Support for all Image Sizes and a Portrait Viewport.

All image sizes are supported; and if the image is too small, its natural resolution is preserved. Viewing size is 300x500 (portrait). Background-color is of a green hue.

Support for Flickr API using JSONP

Flickr API was used with the keyword 'dog' to fetch the images, and the data was in JSONP instead of JSON. Background-color is of a blue hue.
